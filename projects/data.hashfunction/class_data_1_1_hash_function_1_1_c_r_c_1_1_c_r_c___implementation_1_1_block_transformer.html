static IReadOnlyList< UInt64 > Data.HashFunction.CRC.CRC_Implementation.BlockTransformer.GetDataDivisionTable ( int hashSizeInBits,
UInt64 polynomial,
bool reflectIn )
staticprivate

Calculates the data-division table for the CRC parameters provided.

Parameters
hashSizeInBitsLength of the produced CRC value, in bits.
polynomialDivisor to use when calculating the CRC.
reflectInIf true, the CRC calculation processes input as big endian bit order.
Returns
Array of UInt64 values that allows a CRC implementation to look up the result of dividing the index (data) by the polynomial.

Resulting array contains 256 items if settings.Bits >= 8, or 2 items if settings.Bits < 8. The table accounts for reflecting the index bits to fix the input endianness, but it is not possible completely account for the output endianness if the CRC is mixed-endianness.
