Detailed Description

Defines a configuration for a IPearson implementation.

Specifically contains a configuration for Pearson hashing using the lookup table given in the article at http://en.wikipedia.org/wiki/Pearson_hashing as of 2014-04-21.

There is nothing special about this configuration's lookup table that makes it explicitly better or worse than other valid lookup tables.
